# [SscExcelAddIn](https://github.com/standstonecraft/SscExcelAddIn)

自分の業務効率化のための、小さなExcelアドインです。  

## 動作確認環境

- Windows 10 Home 64bit
- Microsoft Excel 2019 MSO Version 2110 Build 16.0.14527.20270 64bit
- .NET Framework 4.8
